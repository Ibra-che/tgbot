# Telegram AI-Бот с ИИ, TTS и админ-панелью

Простой и мощный Telegram-бот с интеграцией ИИ (через OpenRouter), синтезом речи (TTS), профилем, банами, идеями и админ-командами.

### Возможности

- **/ai** — общение с ИИ (GPT, Claude, Gemini, Llama, Qwen и другие через OpenRouter)  
  Пример: `/ai Привет, как дела?`
- **/tts** — текст в речь (мужской или женский голос)  
  Пример: `/tts man Привет, как дела?`
- **/profile** или **/me** — посмотреть свой профиль
- **/suggest** — отправить идею или баг-репорт  
  Пример: `/suggest Добавьте поддержку картинок`
- **Админ-команды** (только для владельца):
  - `/logs` — все логи
  - `/ideas` — все предложения пользователей
  - `/allbans` — список забаненных
  - `/allusers` — первые 15 пользователей
  - `/initbd` — (пере)создать базу данных
  - `/ban` и `/unban` — управление банами (реплай на сообщение)
  - `/newsite` — добавить сайт в список
  - `/site` — показать все сохранённые сайты

### Как запустить

1. **Создай бота**  
   Напиши [@BotFather](https://t.me/botfather) → /newbot → скопируй **токен**

2. **Получи API-ключ OpenRouter**  
   Зарегистрируйся → https://openrouter.ai → Settings → API Keys → создай ключ

3. **Узнай свой Telegram ID**  
   Напиши [@userinfobot](https://t.me/userinfobot) → он покажет твой ID

4. **Создай файл `.env` в корне проекта** и вставь туда:
